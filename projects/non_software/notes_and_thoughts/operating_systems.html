High-Level Summary

The operating system is low-level software that manages the interaction between processes and system resources. Management is necessary to provide:

  • Fairness
    • Regulated and fair resource utilization among competitive processes
  • Security
    • Malicious processes cannot easily damage system resources and other processes
  • Isolation
    • One process malfuctioning does not affect the others
  • Cooperation
    • Multiple processes can be synchronized and communicate with one another

Operating systems define an API for processes to access system resources. Internally, system resources interface with the operating system via device drivers.
